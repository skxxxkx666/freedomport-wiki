# 欢迎使用自由港机场

**自由港机场（FreedomPort）** 提供全球高速网络加速服务。这本手册帮助你从购买到连接，5 分钟完成配置。

## 🚀 我是新用户

按顺序完成三步即可使用：

1. **[快速开始](guide/getting-started.md)** — 注册账号、购买套餐
2. **[订阅链接使用说明](guide/subscription.md)** — 获取你的专属订阅链接
3. **选择你的设备教程** — 按平台完成客户端配置：

| 设备 | 推荐客户端 | 教程 |
| --- | --- | --- |
| 🪟 Windows | Clash Verge Rev | [查看教程](clients/windows/clash-verge.md) |
| 🍎 macOS | Clash Verge Rev | [查看教程](clients/macos/clash-verge.md) |
| 📱 iOS / iPadOS | Shadowrocket | [查看教程](clients/ios/shadowrocket.md) |
| 🤖 Android | Clash Meta for Android | [查看教程](clients/android/clash-meta.md) |
| 📡 路由器 | OpenClash | [查看教程](clients/router.md) |
