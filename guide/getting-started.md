# 快速开始

从零开始使用自由港机场，只需要三步：**注册 → 购买 → 导入**。全程约 5 分钟。

## 第一步：注册账号

1. 打开官网 [freedomport.cc](https://freedomport.cc)，点击右上角 **账号设置** 或直接进入[登录页](https://freedomport.cc/#/auth)
2. 切换到 **注册** 标签，填写邮箱和密码
3. 如果朋友给了你邀请码，在注册页填入（也可以通过邀请链接打开，会自动填好）

{% hint style="info" %}
也可以**先购买后注册**：在[购买页](https://freedomport.cc/#/purchasing)直接以游客身份下单，支付成功后系统会自动创建账号，并把登录信息发送到你填写的邮箱。
{% endhint %}

## 第二步：购买套餐

1. 进入[购买页](https://freedomport.cc/#/purchasing)，或登录后在会员中心点击 **购买订阅**
2. 根据你的需求选择套餐（每张套餐卡列出了流量、设备数、线路覆盖和适合人群）
3. 选择付款周期，确认订单后选择支付方式完成付款
4. 支付成功后订阅自动开通，订单状态会同步到会员中心

{% hint style="warning" %}
建议以**月付为主**，先确认节点适合你的网络环境。退款条件见[使用规范与退款政策](../policy/terms.md)。
{% endhint %}

## 第三步：导入客户端

1. 登录[会员中心](https://freedomport.cc/#/dashboard)，在 **订阅链接** 卡片复制你的专属订阅链接（或直接扫二维码 / 点客户端图标一键导入）
2. 按你的设备打开对应教程，几分钟完成配置：

| 设备 | 推荐客户端 | 备选 |
| --- | --- | --- |
| 🪟 Windows | [Clash Verge Rev](../clients/windows/clash-verge.md) | [v2rayN](../clients/windows/v2rayn.md) |
| 🍎 macOS | [Clash Verge Rev](../clients/macos/clash-verge.md) | — |
| 📱 iOS / iPadOS | [Shadowrocket](../clients/ios/shadowrocket.md) | [Clash Mi](../clients/ios/clash-mi.md) |
| 🤖 Android | [Clash Meta for Android](../clients/android/clash-meta.md) | [v2rayNG](../clients/android/v2rayng.md) |
| 📡 路由器 | [OpenClash 等](../clients/router.md) | — |

3. 客户端里选择节点并开启连接，打开浏览器验证是否可以访问外网
