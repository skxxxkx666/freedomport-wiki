# 常见问题 FAQ

## 连接问题

### 导入订阅后没有节点？

按顺序排查：

1. 在客户端里**手动更新一次订阅**（导入瞬间可能未拉取成功）
2. 确认账号**套餐有效**：会员中心查看到期时间和剩余流量
3. 确认订阅链接复制完整（以 `https://` 开头，中间无空格换行）
4. 换一个网络环境（如切到手机热点）再更新一次

### 节点全部超时 / 无法连接？

1. 更新订阅获取最新节点
2. 换 2–3 个不同地区的节点分别测试（个别节点可能临时维护）
3. 检查本机防火墙 / 杀毒软件是否拦截了客户端
4. 重启客户端，确认"系统代理"已开启
5. 以上都无效 → [提交工单](https://freedomport.cc/#/ticket)，附上客户端日志截图

### 连接成功但网页打不开？

* Clash 系确认模式为 **规则（Rule）** 或 **全局（Global）**，且系统代理开关已打开
* 浏览器如果装了代理插件（SwitchyOmega 等），检查是否冲突
* 尝试清除浏览器 DNS 缓存或更换浏览器验证

### 速度慢？

* 优先选**低延迟**节点（客户端内测速排序）
* 高峰时段（晚 8–11 点）跨境链路普遍拥挤，可尝试不同地区节点
* 大流量下载建议走香港 / 日本等近距离节点

## 账号与套餐

### 流量怎么统计？多久刷新？

上行 + 下行都计入流量。用量数据会回传到会员中心面板，通常几分钟内刷新。

### 设备数限制是什么意思？

指**同时在线**的设备数量。超出限制时新设备会连不上，断开旧设备即可。

### 套餐到期 / 流量用完会怎样？

连接会停止，但账号和订阅链接保留。续费或购买流量后**更新订阅**即可恢复，无需重新配置。

### 可以退款吗？

月度套餐满足条件可退，详见[使用规范与退款政策](../policy/terms.md)。

## 客户端选择

### 我该用哪个客户端？

看[快速开始](getting-started.md#第三步导入客户端)的推荐表。简单说：桌面端用 **Clash Verge Rev**，iOS 用 **Shadowrocket**，Android 用 **Clash Meta**。

### Shadowrocket 为什么下载不了？

它是 App Store **付费应用且国区未上架**，需要美区等外区 Apple ID 购买下载。详见 [Shadowrocket 教程](../clients/ios/shadowrocket.md)。

## 其他

### 担心 DNS 泄露？

Clash 系客户端开启 **TUN 模式**（或"DNS 劫持"）可以避免大多数 DNS 泄露场景；对隐私要求高的用户建议配合浏览器隐私模式使用。

### 还是解决不了？

[提交工单](https://freedomport.cc/#/ticket)（描述设备、客户端、节点和报错现象），或通过 [Telegram 支持](../support.md)联系我们。
