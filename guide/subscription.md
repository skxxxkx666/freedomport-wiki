# 订阅链接使用说明

订阅链接是你账号的"节点通行证"：客户端通过它自动拉取节点列表和分流配置。**一条链接，所有设备通用**（受套餐设备数限制）。

## 在哪里获取

登录[会员中心](https://freedomport.cc/#/dashboard)，**订阅链接** 卡片提供三种方式：

| 方式 | 适用场景 |
| --- | --- |
| **复制链接** | 粘贴到任意客户端的"URL 导入 / 订阅"入口 |
| **扫描二维码** | 手机客户端（Shadowrocket / Clash 系）扫码直接导入 |
| **一键导入** | 点客户端图标，自动唤起本机已安装的客户端完成导入 |

## 更新订阅

节点有增减（例如新地区上线）时，**在客户端里手动更新一次订阅**即可获取最新节点：

* Clash 系：配置页 → 更新（刷新图标）
* Shadowrocket：首页订阅条目左滑 → 更新
* v2rayN / v2rayNG：订阅分组 → 更新订阅

建议每隔一段时间或看到节点上新公告后更新一次。

## 重置订阅

如果怀疑订阅链接泄露（例如发在了公开场合），立即到 **会员中心 → 账号设置 → 订阅中心** 点击 **重置订阅**：

{% hint style="danger" %}
重置后**旧链接立刻失效**，所有设备需要用新链接重新导入。
{% endhint %}

## 安全提醒

* 订阅链接等同于你的账号使用权，**不要分享给他人**，不要发到群聊、论坛、截图中
* 多人使用会触发设备数限制，导致你自己无法连接
* 想邀请朋友使用，请用[会员中心的邀请链接](https://freedomport.cc/#/invite)，你还能拿佣金

## 常见疑问

**Q：订阅链接会变吗？**
A：不会主动变。只有你手动"重置订阅"后才会更换。续费、换套餐都不影响链接。

**Q：一条链接可以几台设备用？**
A：链接本身通用，同时在线设备数以套餐说明为准。

**Q：客户端提示订阅无效 / 拉取失败？**
A：先确认套餐未过期、流量未用完；再检查链接是否复制完整；仍失败请看 [FAQ](faq.md) 或[提交工单](https://freedomport.cc/#/ticket)。
