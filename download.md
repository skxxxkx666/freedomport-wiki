# 客户端下载

所有平台的客户端下载入口汇总。下载后按对应教程完成配置。

::: info
GitHub 直链下载缓慢时，使用带 **镜像加速** 标记的链接。
:::

#### Windows

**Clash Verge Rev**（推荐） · [教程](clients/windows/clash-verge.md)

* [Windows x64 直链](https://github.com/clash-verge-rev/clash-verge-rev/releases/download/v2.4.7/Clash.Verge_2.4.7_x64-setup.exe)
* [Windows x64 镜像加速](https://gh.xxooo.cf/https://github.com/clash-verge-rev/clash-verge-rev/releases/download/v2.4.7/Clash.Verge_2.4.7_x64-setup.exe)
* [全部版本（GitHub Releases）](https://github.com/clash-verge-rev/clash-verge-rev/releases)

**v2rayN** · [教程](clients/windows/v2rayn.md)

* [GitHub Releases](https://github.com/2dust/v2rayN/releases)

#### macOS

**Clash Verge Rev**（推荐） · [教程](clients/macos/clash-verge.md)

* [Apple Silicon（aarch64）Releases](https://github.com/clash-verge-rev/clash-verge-rev/releases)
* [Intel（x64）Releases](https://github.com/clash-verge-rev/clash-verge-rev/releases)
* 下载 `.dmg` 后拖入应用程序目录即可

#### iOS / iPadOS

**Clash Mi**（推荐 · 免费） · [教程](clients/ios/clash-mi.md)

* [App Store 直达](https://apps.apple.com/app/id6744321968)

**Shadowrocket**（备选 · 付费，需外区 Apple ID） · [教程](clients/ios/shadowrocket.md)

* [App Store（美区）](https://apps.apple.com/us/app/shadowrocket/id932747118)

#### Android

**Clash Meta for Android**（推荐） · [教程](clients/android/clash-meta.md)

* [GitHub Releases](https://github.com/MetaCubeX/ClashMetaForAndroid/releases)

**v2rayNG** · [教程](clients/android/v2rayng.md)

* [GitHub Releases](https://github.com/2dust/v2rayNG/releases)

#### 更多

**Linux · Clash Verge Rev** · [教程](clients/others/linux.md)

* [GitHub Releases](https://github.com/clash-verge-rev/clash-verge-rev/releases)（.deb / .rpm / AppImage）

**路由器 · OpenClash** · [教程](clients/others/router.md)

* [OpenClash Releases](https://github.com/vernesong/OpenClash/releases)

::: warning
请只从上述官方渠道下载客户端，第三方渠道的安装包可能被植入恶意代码。
:::

下载完成后，回到[快速开始](guide/getting-started.md)完成订阅导入。
